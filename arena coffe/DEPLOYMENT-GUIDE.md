## How to Make Your Website Live and Public

### Option 1: GitHub Pages (FREE & Professional)

#### Step 1: Create GitHub Account
1. Go to [github.com](https://github.com)
2. Sign up for a free account
3. Verify your email

#### Step 2: Create Repository
1. Click "New repository"
2. Name it: `arena-coffee`
3. Set to Public
4. Don't initialize with README
5. Click "Create repository"

#### Step 3: Upload Your Files
1. Download all files from your project folder
2. On GitHub, click "Add file" â†’ "Upload files"
3. Drag and drop these files:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`

#### Step 4: Enable GitHub Pages
1. Go to repository "Settings"
2. Scroll to "Pages" section
3. Under "Source", select "Deploy from branch"
4. Choose "main" branch
5. Click "Save"
6. Wait 1-2 minutes for deployment

#### Step 5: Get Your Live URL
Your website will be available at:
`https://YOURUSERNAME.github.io/arena-coffee`

Example: `https://johnsmith.github.io/arena-coffee`

---

### Option 2: Netlify (Easiest Drag & Drop)

#### Step 1: Prepare Your Files
1. Create a folder named `arena-coffee`
2. Put all website files inside this folder

#### Step 2: Deploy to Netlify
1. Go to [netlify.com](https://netlify.com)
2. Sign up/Login with GitHub
3. Drag your `arena-coffee` folder to the upload area
4. Wait for automatic deployment
5. Get your custom URL like: `your-site.netlify.app`

---

### Option 3: Vercel (Professional Alternative)

#### Step 1: Install Vercel
1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub
3. Install Vercel CLI or use web interface

#### Step 2: Deploy
1. Import your GitHub repository
2. Connect to Vercel
3. Deploy with one click
4. Get professional URL
